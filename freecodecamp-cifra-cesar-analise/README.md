# Cifra de César com Análise de Frequência

## 📌 Descrição
Este projeto implementa a cifra de César e realiza uma análise de frequência das letras após a criptografia.

## ⚙️ Funcionalidades
- Criptografia de texto
- Análise de frequência de letras
- Visualização com gráfico

## ▶️ Como executar

```bash
python main.py
